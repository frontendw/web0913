position

  • 요소에 사용되는 위치 지정 방법의 유형을 지정한다.
  • 속성 값은 static(정적, 기본값), relative(상대), absolute(절대), fixed(고정), sticky(고정)가 있다.
  • position:static;이 아니면 top, left, right, bottom, z-index 속성을 사용할 수 있다.
  • relative는 원래 위치(자기 자신)를 기준으로 left, top, right, bottom이 적용된다.
  • absolute는 static이 아닌 position 속성을 지정한 조상가 없을 경우 뷰포트를 기준으로 left, top, right, bottom이 적용된다.
  • absolute는 static이 아닌 position 속성을 지정한 가장 가까운 조상를 기준으로 left, top, right, bottom이 적용된다.
  • fixed는 뷰포트를 기준으로 left, right, top, bottom이 적용된다.
  • relative는 원래 위치는 그대로 유지된다.
  • absolute와 fixed는 원래 위치가 유지되지 않는다. 즉, 아래쪽 요소가 위로 올라온다.!!!!
  • fixed는 스크롤 시 화면에 고정된다.
  • 크기를 지정하지 않으면 static과 relative의 width는 100%이고, absolute와 fixed는 요소의 내용 크기로 지정된다.
  • z-index는 스택(stack) 순서를 지정한다.
  • z-index의 값은 정수로 숫자가 클수록 앞으로 표시된다.
  • z-index를 사용하지 않을 경우 아래 기록한 요소가 앞으로 표시된다.
  • static 이면 z-index를 지정할 수 없으므로 static이 아닌 경우 보다 뒤에 있다. 그러나, z-index를 음수로 지정하면 static이 아닌 경우도 static 보다 뒤로 갈 수 있다.
